Primitive Metric Types

For more detailed information on building metrics see the Metrics DSL section.

The examples in this section assume you have imported the following and have created a MetricFactory:

import cats.effect._
import prometheus4cats._

val factory: MetricFactory[IO] = MetricFactory.noop[IO]
// factory: MetricFactory[IO] = prometheus4cats.MetricFactory$$anon$36@70c1d76c

Counter

factory.counter("my_counter_total").ofLong.help("Metric description")
// res0: internal.MetricDsl[IO, Long, Counter, Counter.Labelled] = prometheus4cats.internal.MetricDsl@7d3a37a4

Gauge

factory.gauge("my_counter_total").ofLong.help("Metric description")
// res1: internal.MetricDsl[IO, Long, Gauge, Gauge.Labelled] = prometheus4cats.internal.MetricDsl@29a3367d

Histogram

factory.histogram("my_counter_total").ofDouble.help("Metric description").buckets(0.1, 0.5, 1.0, 1.5, 2.0)
// res2: internal.MetricDsl[IO, Double, Histogram, Histogram.Labelled] = prometheus4cats.internal.MetricDsl@5e2805f9
